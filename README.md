# Sunset

# 更名 Sunset



### 简介

A gift.
一个能够进行人机交互的桌面摆件。

- 交互方式：按键，姿态，语音（或加入手势控制）。

### 硬件

- ESP32-PICO-D4：蓝牙，wifi控制（或接入网络）。主要实现
- 1.44屏：交互显示
- 拨码按键：按键交互
- 是否需要开关按键 ？
- 触摸功能，电容触摸键
- RGB灯光：布景和交互提示
- 特别版加入分光棱镜
- 磁吸连接供电/无线供电 ？
- MPU6050或其他姿态感知元件
- 是否添加光电传感器 ？

### 软件

- 预计语言：C、Python、C++
- Fusion360构建模型
- AD：绘制电路板
- VSCode：代码编写
- Anaconda：环境搭建 

### 外观设计

- LVGL设计
- 各功能分配，芯片配合
- 考虑是否搭建网站来独立进行信息输入输出
- 供电选择
- 美观性，外形设计
