# 550S

### 简介

一个能够进行人机交互的桌面摆件。

- 交互方式：按键，姿态，语音（或加入手势控制）。

### 硬件

- ESP32：蓝牙，wifi控制（或接入网络）。
- K210：音频处理（或加入图像处理）。
- ESP32/K210：再使用一枚芯片独立LVGL实现
- 1.44屏：交互显示
- 数字麦克风：获取音频
- 拨码按键：按键交互
- RGB灯光：布景和交互提示
- 特别版加入分光棱镜

### 软件

- 预计语言：C、Python、C++
- Fusion360构建模型
- AD：绘制电路板
- VSCode：代码编写
- Anaconda：环境搭建

### 设计难点

- 音频处理和模型训练获取
- LVGL设计
- 芯片选择
- 各功能分配，芯片配合
- 考虑是否搭建网站来独立进行信息输入输出
- 供电选择
- 美观性，外形设计
